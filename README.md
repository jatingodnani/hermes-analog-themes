# Hermes Analog Theme Pack 🎨

A collection of 4 highly customized, immersive themes for the Hermes Agent Dashboard. Designed to bring physical, analog, and retro aesthetics to your AI agent interface.

## The Themes

### 1. Manuscript 🖋️
A warm, aged-paper aesthetic with deep brown typography and elegant gold accents. Focuses on an organic, analog feel like reading a high-quality journal.
* **Vibe:** Academic, organic, classical.

![Manuscript 1](./assets/manuscript/sc-1.png)
![Manuscript 2](./assets/manuscript/sc-2.png)
![Manuscript 3](./assets/manuscript/sc-3.png)

### 2. Paper 📐
A structural, drafting-paper design with subtle ruled lines, architectural spacing, and rust/green accents. Clean and typographic.
* **Vibe:** Engineering, drafting, structural.

![Paper 1](./assets/paper/sc-1.png)
![Paper 2](./assets/paper/sc-2.png)
![Paper 3](./assets/paper/sc-3.png)

### 3. CRT Amber 📺
A faithful recreation of vintage monochrome CRT monitors. Features amber phosphors, deep black backgrounds, and subtle scanline textures.
* **Vibe:** Retro-computing, cyberpunk, terminal.

![CRT Amber 1](./assets/ctr-amber/sc-1.png)
![CRT Amber 2](./assets/ctr-amber/sc-2.png)
![CRT Amber 3](./assets/ctr-amber/sc-3.png)

### 4. CRT Green 📟
The classic green-phosphor terminal aesthetic. High contrast, aggressive glow, and undeniable hacker vibes.
* **Vibe:** Mainframe, matrix, command-line.

![CRT Green 1](./assets/ctr-green/sc-1.png)
![CRT Green 2](./assets/ctr-green/sc-2.png)
![CRT Green 3](./assets/ctr-green/sc-3.png)

## Installation
1. Clone or download this repository.
2. Open your Hermes Dashboard.
3. Navigate to **Config -> Appearance**.
4. Click **Import Theme** and select any of the `.yaml` files from this pack.
